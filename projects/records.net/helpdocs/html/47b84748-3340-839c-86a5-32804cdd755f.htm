Upstream.System.Records Namespace

Provides core interfaces with some basic implemention classes for objects that manipulate data records consisting of named fields.
Classes
  ClassDescription
ArrayRecordListTValue, TFieldType
Implements a record list on a list of record field value arrays
BasicRecordFieldTypeTValue
Provides the obvious implementation for IRecordFieldProperties
BasicRecordSchemaTFieldType
Implements IRecordSchemaAccessor in a very basic way.
DictionaryRecordAccessorTValue, TFieldType
Implements IRecordAccessor over a .Net Dictionary
FieldSchemaSpecTValue
Provides methods for parsing a field schema spec mini language
FieldSchemaSpecFieldRecordTValue
Provides access to meta-fields for a field parsed from a Field Schema Spec string.
ListRecordAccessorTValue, TFieldType
Provides a base record accessor implemented on basic arrays
ParsingRecordAccessorTFieldType
Adapts from a string record accessor to an object record accessor by parsing datatypes according to Convert.ChangeType() (which uses IConvertible)
PrintingRecordAccessorTValue, TFieldType
Implements IRecordAccessorAdapter to convert field values from a base record into string values
RecordEnumeratorAdapterTValue, TBaseRecord
Implements IRecordEnumerator over a base enumerator using a record adapter
RecordIO
Provides some helper methods for copying data between records
RecordListIndexTValue, TFieldType
Implements an IRecordCollectionIndex over an IRecordList. The index cannot detect changes to the underlying list, so if the list is changed, the index must be updated separately.
Interfaces
  InterfaceDescription
IRecordAccessorTValue
Defines a basic field-value lookup interface for tabular data records.
IRecordAccessorAdapterTValue, TBaseRecord
Defines a record accessor that gets its values from some base object
IRecordCollectionTValue, TField
Defines a collection type for record accessors. Doesn't implement ICollection, but implements a subset of it. Enumerating this collection does not necessarily return distinct objects since an IRecordAccessor is a "visitor".
IRecordCollectionBuilderTValue
Defines a collector-type interface that provides a working-record that can be written-back to a record collection
IRecordCollectionIndexTValue
Defines an interface for a searchable index on an underlying record collection.
IRecordCollectorTValue
Implements a "collector" object which can add records to a collection
IRecordEnumeratorTValue
Defines an enumerator-like interface for stepping over a collection of records. Implements an enumerator interface, but the "current" object is assumed to be an unchanging reference to a "visitor" object which changes its state as the enumerator moves.
IRecordFieldTypeTValue
Defines a basic interface which RecordFieldProperties implementations should implement
IRecordListTValue, TField
Defines a collection type for record accessors. Doesn't implement IList, but implements a subset of it. Enumerating this collection does not necessarily return distinct objects since an IRecordAccessor is a "visitor".
IRecordListVisitorTValue
Provides an efficient, enumerator-like "cursor" with random-access to a record list.
IRecordSchemaAccessorTFieldType
Defines an object that provides access to the field metadata for a record collection
