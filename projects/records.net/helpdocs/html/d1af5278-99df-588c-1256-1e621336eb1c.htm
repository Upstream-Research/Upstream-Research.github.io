Upstream.System.Csv Namespace

Provides basic objects for reading and writing general "Code Separated Value" format - which includes the familiar "Comma Separated Value" and Tab delimited formats as a specific cases.
Classes
  ClassDescription
CsvEncoding
Basic Code Separated Value (CSV) encoder/decoder object
CsvException
Basic Exception for CSV components
CsvReader
Implements a stream-reader type object for parsing a Code-Separated-Value text stream
CsvWriter
Implements a TextWriter wrapper for writing CSV "records" consisting of "unit" values. Records are delimited by the newline supported by the underlying text writer.
