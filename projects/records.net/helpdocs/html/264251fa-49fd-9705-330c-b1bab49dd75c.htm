Remarks

This interface implements IEnumerable{IFieldNameValuePair{V}}. IDictionary{K,V} implements IEnumerable{KeyValuePair{K,V}}. The use of the IFieldNameValuePair{V} interface instead of the KeyValuePair{K,V} value type (struct) is to allow covariance in the value type parameter (KeyValuePair{K,V} is not covariant).
