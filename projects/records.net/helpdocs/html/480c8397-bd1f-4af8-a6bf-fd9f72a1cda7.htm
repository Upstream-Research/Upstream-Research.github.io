Records.Net - Data Record Framework

Records.net is an object library for manipulating record-based data. It provides a set of data record interfaces that can be used as a basis for record I/O streams and for record-based data models.

NOTE: This documentation is under development, please excuse the presence of place-holders and "TODO" items.

This library provides an alternative to some of the data structures and interfaces in Microsoft's ADO.Net (Aka. System.Data). It does not provide any means to communicate with a database.

This library is not an Object/Relational Mapper. The fundamental point of view taken here is that records are themselves a data structure that should not be mapped onto "plain old" objects. There are inconvenient consequences to this approach, but this library is provided for those who are willing to accept the consequences, perhaps for the promise of other efficiencies.
